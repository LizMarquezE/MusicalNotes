Kramer Pariah – Sonido poderoso con estilo profesional

La Kramer Pariah es una guitarra eléctrica moderna, diseñada para guitarristas que buscan potencia, versatilidad y estética impactante. Su acabado Candy Red brilla bajo los focos del escenario.

Incluye sistema Push Pull que permite dividir las pastillas humbucker para obtener sonidos tipo single coil, brindando una paleta sonora mucho más amplia.

Características de la guitarra Kramer Pariah:

  • Color: Candy Red brillante
  • Pastillas humbucker con sistema Push Pull
  • Cuerpo sólido ergonómico
  • Mástil cómodo para técnica rápida
  • Ideal para escenarios y estudio profesional
